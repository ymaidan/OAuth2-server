What happens when you click Login?

  1. Browser redirects to /login on this app
  2. App redirects you to OAuth /authorize
  3. Mock server auto-approves and redirects to /callback?code=...
  4. App exchanges the code for an access_token
  5. App calls /userinfo and displays the results
